This is the first post on the blog.

## Why a blog?

I've been building software for years — full-stack web applications, mainframe systems, AI tooling — and I've always wanted a space to write about the things I learn along the way.

This blog is that space. Expect posts about:

- **Software engineering** — architecture, patterns, and tools
- **Mainframes** — COBOL, JCL, and the systems that still run the world
- **Side projects** — the things I build for fun

## The stack

This blog is built with [UtopiaJS](https://github.com/wrxck/utopiajs), my own frontend framework. The content is authored in Markdown, rendered at build time, and served as a static SPA.

No JavaScript frameworks were harmed in the making of this blog. Well, one was — I built it.
